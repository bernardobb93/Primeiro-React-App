# Meu primeiro projeto com React

## Scripts

Para instalar o npm:

### `npm install`

Iniciar a aplicação:

### `npm start`

Visualizar a aplicação

### `Abrir` [http://localhost:3000](http://localhost:3000)

Para criar a aplicação

### `npm run build`
